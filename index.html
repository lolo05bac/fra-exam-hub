How to use this site

  1. Start with the Chapters. Read each one twice. Pay particular attention to Memory Hooks — that's how you'll recall formulas under pressure.
  2. Drill Practice per chapter until you score ≥80% comfortably. Mark anything you got wrong as "weak".
  3. Do Mock Exam 1 cold — even before you feel ready. Find out what 90 minutes feels like.
  4. Use Master Quiz in "weak spots only" mode every day in the final week.
  5. Take Mocks 2-4 in the final five days, timed, no notes. Treat each one like the real exam.
